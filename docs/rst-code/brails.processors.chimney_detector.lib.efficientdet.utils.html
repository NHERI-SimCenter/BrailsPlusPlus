forward(image, dtype=torch.float32)

Generates multiscale anchor boxes.

Parameters:

  • image_size – integer number of input image size. The input image has the same dimension for width and height. The image_size should be divided by the largest feature stride 2^max_level.

  • anchor_scale – float number representing the scale of size of the base anchor to the feature stride 2^level.

  • anchor_configs – a dictionary with keys as the levels of anchors and values as a list of anchor configuration.

Returns:

a numpy array with shape [N, 4], which stacks anchors on all

feature levels.

Return type:

anchor_boxes

Raises:

ValueError – input size must be the multiple of largest feature stride.
